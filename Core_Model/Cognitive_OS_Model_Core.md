思考アルゴリズム骨格論
Cognitive OS Dynamic Structure Model
Overview

思考アルゴリズム骨格論（Skeleton Theory）は、

認知アルゴリズムの状態変化と構造更新過程を観測・モデル化するための理論体系

である。

本理論は人格分類や能力評価を目的としない。

対象は、

注意
感情
思考
意味生成
行動

が環境との相互作用によって変化する動的構造である。

Philosophy

骨格論の基本思想は、

人を分類するためではなく、人が自身の認知構造を理解し更新するためのモデルを提供する

ことである。

人間の認知は固定された性質ではなく、

環境・目的・経験・状態によって変化する動的システムとして扱う。
